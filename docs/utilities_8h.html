template<typename InputType>
InputType get_value_in_range(const InputType& min, const InputType& max, const std::string& prompt, const std::string& error)

Get a value in a range.

Template parameters
InputType The type of object that must be inserted.
Parameters
min in The minimum value that can be accepted _(inclusive)_.
max in The maximum value that can be accepted _(inclusive)_.
prompt in The message that will be displayed to ask the user for an input.
error in The message that will be displayed in case of errors.
Returns The inputed value that is in the range [min, max].

Given a range, this functions prompt the user for an input. In case of error (if the inserted value isn't in the range or other kind of errors that can set to true the std::cin.fail() flag), the user is asked to insert a new value.

Note

The error message is prepended to the prompt value with the only addition of a space between the two values. So the resulting message will be like

error + " " + prompt

Precondition

In order to use this function on all kinds of object, there must be an ovveride of the operator <, operator > and operator >> (std::istream&, InputType&).
